# Godot Minigame 1
## Devlog
One way I think about the relationship between nodes and scenes is like a garden. A scene is almost like the entire garden, while the Nodes are the individual flowers, animals, and seeds inside of it. Each object has its own purpose, but when they are placed together, they create a single and complete environment. This analogy works because Nodes are the smaller pieces that make up the bigger structure of a scene. A garden needs different flowers, vegetables, and fruits to be labeled as a garden and to function, which is similar to how this scene needs different nodes like platforms, coins, springs, and bombs to create a real gameplay.
